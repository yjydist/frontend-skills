# Frontend Skills Extension

本扩展提供两套前端 UI 设计技能, 用于创建风格鲜明的生产级 Web 界面.

## Skills

### factory-industrial-web
工业蓝图风格: 灰色斜纹纸背景, 1px 纯黑硬边, 无圆角, 全大写标签, 纯黑按钮, 单一橘色强调色(#F06D2D).

**适用场景**: 用户明确请求工业风, 蓝图, 工程图, 硬派 (brutalist), 工厂, 技术文档, 斜纹纸等美学风格, 用于创建设置页, 管理面板, 仪表盘, 登录页, 营销站, 博客, 作品集, 电商页, 状态页, 开发者控制台等.

### retro-pixel-web
复古打字机风格: 米色背景, 粗黑边框, 紫色/靛蓝强调色(#6366F1), 全等宽字体 (Courier Prime / Sarasa Mono SC), 卡片式布局.

**适用场景**: 用户明确请求复古, 怀旧, 打字机, 像素风格, 极客, 终端风等美学风格, 用于创建网页, 登录页, 博客, 作品集, 电商站, 管理面板, 仪表盘等.
